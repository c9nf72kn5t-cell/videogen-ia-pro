# VidéoGen IA Pro

**Le meilleur générateur de vidéos IA open-source** - Text-to-Video, Avatars, Templates pro comme InVideo & Synthesia.

## Fonctionnalités Incroyables
- Templates professionnels
- Voix off réalistes (français)
- Génération d'images IA
- Avatars parlants
- Sous-titres, musique, effets
- Export MP4 haute qualité
- Interface desktop native

## Installation
1. `npm install`
2. `npm start`

Pour build : `npm run dist`
